# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to us privately. We will review the issue and work with you to resolve it as quickly as possible.

### How to Report

1. **Email us:** Send an email to ghorbanimahdi565@gmail.com with the details of the vulnerability.
2. **Provide details:** Include as much information as possible to help us understand and reproduce the issue. This may include:
   - Steps to reproduce
   - Affected versions
   - Potential impact

### Response Time

We aim to respond to all security reports within **48 hours**. Once we have reviewed and verified the issue, we will:

- Provide an initial response with our assessment.
- Work on a fix and provide updates on the progress.
- Notify you when the issue is resolved and a patch is released.

## Security Updates

We will announce security updates in the following ways:

- **GitHub Releases:** Details of the fix will be included in the release notes.
- **Changelog:** The changelog will highlight security-related fixes.

Thank you for helping us keep ForgeLoop secure!
