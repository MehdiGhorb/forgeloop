# Contributing to ForgeLoop

We welcome contributions to ForgeLoop! Here’s how you can get involved:

## How to Contribute

1. **Report Issues**: Found a bug or have a feature request? Open an issue in the repository.
2. **Submit Pull Requests**: Fix a bug, add a feature, or improve documentation. Fork the repository, make your changes, and submit a pull request.
3. **Review Code**: Help review open pull requests to ensure quality and consistency.

## Development Setup

1. Clone the repository.
2. Install dependencies using `bun install`.
3. Run the development server with `bun dev`.

## Code of Conduct

Please follow our [Code of Conduct](CODE_OF_CONDUCT.md) to ensure a welcoming environment for everyone.

Thank you for contributing to ForgeLoop!
