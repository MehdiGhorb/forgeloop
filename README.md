# ForgeLoop - Autonomous AI Coding Agent

ForgeLoop is an autonomous AI coding agent that runs 24/7. It can push to GitHub on its own, works on the code as long as needed to fully build it without your intervention, and can test at a user level (for example, with Selenium) to make sure the app is functional.

This project was cloned from OpenCode and built upon: https://github.com/anomalyco/opencode
