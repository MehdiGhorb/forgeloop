# ForgeLoop - Autonomous AI Coding Agent

ForgeLoop is an autonomous AI coding agent that runs 24/7. It can push to GitHub on its own, works on the code as long as needed to fully build it without your intervention, and can test at a user level (e. g., with Selenium).

## Installation

To install ForgeLoop, clone this repository and run the install script:

```bash
bash install
```

Then restart your terminal or run `source ~/.zshrc` and you can start using:

```bash
forgeloop
```

## Uninstallation

To completely uninstall ForgeLoop:

```bash
rm -rf ~/.forgeloop
```
